Mockend

{Mock} End

Free fake API for testing and prototyping.

Coded and maintained with ❤️ by riccardogenova

When to use

Mockend is a free online REST API that you can use whenever you need some fake data. It can be in a README on GitHub, for a demo on CodeSandbox, in code examples on Stack Overflow, ...or simply to test things locally.

Resources

Mockend comes with a set of 6 common resources:
/albums 100 albums
/comments 500 comments
/photos 100 photos
/posts 5000 posts
/products 200 products
/users 10 users

Routes

Actually only some HTTP methods are supported. You have to use https for your requests.
GET /products
GET /products/1
